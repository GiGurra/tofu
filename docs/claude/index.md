# Claude Code Integration (Deprecated)

!!! warning "Moved to a new repo"
    The Claude Code extensions have been extracted into their own repository:

    **[tofutools/tclaude](https://github.com/tofutools/tclaude)** — [Documentation](https://tofutools.github.io/tclaude/)

    Install from the new repo:
    ```bash
    go install github.com/tofutools/tclaude/cmd/tclaude@latest
    ```

The `tofu claude` subcommand and the `cmd/tclaude` binary in this repo are deprecated and will be removed in a future release. All new development happens in the new repo.
