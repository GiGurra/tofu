# `tofu k8s`

Kubernetes utilities.

## Synopsis

```
tofu k8s [command]
```

## Description

A collection of Kubernetes utilities for common operations.

## Commands

### tail

Tail Kubernetes resources (logs, events, etc.).

```
tofu k8s tail [flags]
```

## Examples

Tail Kubernetes resources (see subcommand help for details):

```
tofu k8s tail --help
```
