# `tofu screensaver`

Display an animated tofu bowl screensaver.

## Synopsis

```
tofu screensaver [flags]
```

## Description

Display an animated tofu bowl with chopsticks in your terminal. Press Ctrl+C to exit.

## Options

- `-f, --fps int`: Frames per second (default 10)

## Examples

Start the screensaver:

```
tofu screensaver
```

Run at higher frame rate:

```
tofu screensaver -f 30
```
