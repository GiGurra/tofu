# `tofu gh`

GitHub utilities.

## Synopsis

```
tofu gh [command]
```

## Description

A collection of GitHub utilities for common operations.

## Commands

### list-repos

List repositories for a user, org, or team.

```
tofu gh list-repos [flags]
```

## Examples

List repositories (see subcommand help for details):

```
tofu gh list-repos --help
```
